Fortnite Script Documentation

Current Commands

1. drop - Declares or assigns variables

Syntax: drop variableName with value

drop health with 100
drop name with "Player"

2. shield - Prints/outputs values

Syntax: shield value

shield health
shield "Hello World"
shield 42

3. eliminate - Deletes variables

Syntax: eliminate variableName

eliminate health

4. storm - Conditional statements (partially implemented)

Syntax: storm condition then

storm health > 50 then

Supported Operations

Mathematical Operations:

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
drop damage with 50
drop armor with 0.8
drop final_damage with damage * armor

Syntax Rules

Variable names:

  • Must start with a letter
  • Can contain letters, numbers, and underscores
  • Are case-sensitive

Values can be:

  • Numbers (integers or decimals)
  • Strings (with or without quotes)
  • Mathematical expressions
  • References to other variables

Example Code

// Basic Variables and Math
drop health with 100
drop damage with 25
drop remaining_health with health - damage
shield remaining_health  // Shows 75

// Multiple Operations
drop shield with 50
drop total_protection with health + shield
shield total_protection  // Shows 150

// Variable Management
drop potion with 50
shield potion  // Shows 50
eliminate potion
drop potion with 25
shield potion  // Shows 25
