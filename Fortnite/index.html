Quick Reference:

Commands:
drop [variable] with [value]    - Create a variable (e.g., drop shield_potion with 100)
shield [value]                  - Print a value (e.g., shield "Hello World")
eliminate [variable]            - Delete a variable (e.g., eliminate shield_potion)
storm [condition] then          - If statement (e.g., storm shield_potion > 50 then)

Examples:
drop shield_potion with 100
drop mini_shields with 25
shield shield_potion
eliminate shield_potion
storm mini_shields > 20 then
